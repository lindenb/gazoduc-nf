## Parameters

 * --samplesheet json/csv file containing the following columns: sample,bam,sex (male|female), status (case|control)
 * --fasta reference genome
 * --catalog (file) custom json catalog. Optional
 * --mode (streaming|seeking)
 * --xhunter_args (string) specific parameters for xhunter
 * --prefix (string) custom file prefix. Optional
 * --outdir (dir) custom output directory. Optional
* --keep_bams (boolean) save BAM files
* --keep_json (boolean) save JSON files
